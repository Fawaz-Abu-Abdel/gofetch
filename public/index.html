What gofetch Does

Five Probes. One Command.

Every probe runs concurrently, so you get a complete picture of any target in seconds, not minutes.

DNS Intelligence

Enumerate A, AAAA, CNAME, MX, and TXT records in parallel using Go's net package. Full record inspection in a single lookup.

A · AAAA MX · TXT CNAME

TLS / SSL Inspection

Extract issuer, subject, expiry dates, and SANs via crypto/tls. Get expiry warnings at 14 and 30-day thresholds — never get blindsided by a cert renewal.

Issuer Expiry SANs

GeoIP & ASN Mapping

Query ip-api.com to resolve country, region, city, ISP, and Autonomous System Number for any resolved IP address.

Country ISP ASN

HTTP Header Analysis

Probe all seven critical security headers — HSTS, CSP, X-Frame-Options, and more. Instantly see which are missing and which are configured.

HSTS CSP 7 headers

Concurrent Port Scan

Probe 25 common TCP ports with net.DialTimeout using goroutine-per-port concurrency. Completes in ~1s wall time regardless of port count.

25 ports Concurrent 1s timeout
